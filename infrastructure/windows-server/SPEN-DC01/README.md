# SPEN-DC01

## Overview

SPEN-DC01 is the first Windows Server deployed within the SpenTech Enterprise Infrastructure environment.

It serves as the primary Domain Controller and will provide centralized identity management, DNS, DHCP, and Group Policy services for the organization.

## Specifications

- Azure Region: East US
- VM Size: Standard_B2s
- Operating System: Windows Server 2022 Datacenter: Azure Edition
- Virtual Network: VNET-SpenTech
- Subnet: Servers
